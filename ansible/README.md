# Service Status Checks

- Nginx Status Check
```sh
sudo nginx -t
sudo systemctl status nginx
```

- PHP Status Check
```sh
php -v
php-fpm -t
sudo systemctl status php-fpm.service
```

- Database Login
```sh
mysql -u root -h db -p
```

- Zabbix Status Check
```sh
sudo systemctl status zabbix-agent.service
sudo systemctl status zabbix-server.service
```
